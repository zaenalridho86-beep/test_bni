# Dokumentasi API

## Deskripsi
API ini digunakan untuk ["mengelola data users"].

## Endpoint
Berikut adalah daftar endpoint yang tersedia:

* *GET /users*: menapilkan data users
* *POST /users*: Membuat data user baru
* *PUT /users/{id}*: Memperbarui/update data user
* *DELETE /users/{id}*: Menghapus user

## Request Body
Format request body yang digunakan adalah JSON.

## Response
Format response yang digunakan adalah JSON.
